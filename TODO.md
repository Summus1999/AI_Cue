# 项目进度文档

## 1期工程

1.做一个界面，界面有一个对话框，可以用于和AI进行文字交互。界面是一个蓝色的界面，长条的，比较小，然后有一个带有圆角的对话框，可以输入文字和AI对话。✅️
2.这个界面的对话框不会被腾讯会议捕捉到。✅️
3.界面右上角加一个设置按钮，点击之后能设置各项东西，先支持可以设置用户的模型，当前先默认支持千问模型，然后加一个下拉框，可以选择回答问题的千问哪个模型。✅️
4.主界面增加一个可以语音输入的功能按钮，就在发送按钮的旁边，点击这个语音输入的按钮，就会录入电脑输出的音频。✅️
5.设置栏里增加设置语音识别控制阈值，界面要美观。✅️
6.集成阿里云语音识别 API ✅️
7.增加一个设置项，可以选择语音识别后处理调用的是什么模型✅️
8.实现语音识别后能够进行AI回答的功能✅️
9.在设置里增加一个可以控制prompt的选项，在这个文本框内输入你的prompt，方便AI后续能够很好的回答你的面试问题。✅
10.修改设置界面为另一个界面，点击后跳转到设置界面，点保存返回主界面。✅
11.界面右上角设置的左边增加一个按钮，这个按钮专门负责设置tauri的各个快捷键，如录制音频的快捷键、发送的快捷键。✅
12.让输出的AI的回答显示为正常的格式，而不是markdown格式。✅
13.让AI的回答改为流式输出，而不是一次性直接输出。❌️
